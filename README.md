# Forestry Reserve & Conservation Platform

A modern, high-performance NGO platform dedicated to the preservation of global forest ecosystems.

## Core Mission

Forestry Reserve is focused on restoring old-growth forests, rehabilitating local ecosystems, and fostering deep connections between humanity and the woods through innovative conservation and education programs.

## Key Features

- **Project Tracking**: Detailed overviews of remediation and conservation projects.
- **Urgent Campaigns**: Rapid-response funding for forest crises (wildfires, logging).
- **Live News**: Real-time ecological insights from around the globe.
- **Transparency Portal**: Audited financial reports and ethical compliance.

## Environment & Tech

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion & GSAP
- **State Management**: Zustand
- **CMS/Data**: Mock-integrated for demo-first development
